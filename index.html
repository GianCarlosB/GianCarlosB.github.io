Outras ferramentas de design web

W3C Markup Validation Service

Este validador verifica a validade de marcação de documentos da Web em HTML, XHTML, SMIL, MathML, etc Se quiser validar o conteúdo específico, como feeds RSS / Atom ou folhas de estilo CSS, o conteúdo mobileOK, ou para encontrar links quebrados, existem outros validadores e ferramentas disponíveis.

Browser Shots

Browsershots faz capturas de tela do seu design web em diferentes sistemas operacionais e navegadores. É uma forma simples de testar a compatibilidade entre os navegador do website a sua em um só lugar.

Tidy CSS

CSSTidy é uma fonte aberta CSS analisador e otimizador que irá corrigir erros e otimizar seu código CSS.

Google Font Compare

Basta digitar algumas palavras e ver como será a aparência de diferentes fontes.
